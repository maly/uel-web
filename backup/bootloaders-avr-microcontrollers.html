Bootloaders for AVR microcontrollers

24. April 2008 - 20:13 — admin

A rich list of available bootloaders for AVR microcontrollers.

Quotation: Probably most proper microcontroller programming method is by using bootloader program. This way you don’t need any special programming adapters or special knowledge - you just need to connect standard cable from your PC to target board and run special program which communicates with MCU bootloader program.

Most of AVR bootloaders are using COM port for connectivity with PC. As a mater of fact it is not always to run somebody’s written bootlaoder at first time. Usually it may require some customizations, reading help documents. In the list you can see a list of most popular bootloaders available.
