Megavore

5. December 2009 - 19:05 — admin

The Megavore is an Atmel AVR xMEGA 128A1 development board, similar to Arduino (but not exactly pinout compatible).

If you want to start with a XMega family, try the Megavore board. It features:

* An Atmel AVR xMEGA 128A1, running at up to 32MHz
* A USB to serial adapter, using the venerable FTDI FT232R chip
* All of the pins not in use for crystals and power available on 0.1 inch headers
* All of the normal support parts, including a 3.3V 1 amp linear regulator, power connectors, USB-B connector, etc

The board is Open Source Hardware under the Creative Commons 3.0 Share-Alike Attribution license.
