Raspberry Pi - Single board Linux for 25 USD

18. September 2011 - 8:32 — admin

Wanna have a Linux box (ARM driven, SDHC disc, USB & HDMI+Composite video output) for 25 USD? Try a Raspberry Pi!

Raspberry Pi is an ARM11-based SBC with 128MB RAM, 2xUSB and SDHC card (as disc), powered by Linux - it's all we know. This configuration will cost 25 USD. Configuration B (a slightly better) is for 35USD approx. and it has more RAM (256MB) and built-in Ethernet. Very nice for embedded computing, isn't it?

The first Raspberries will be available at the end of 2011. There is no preorders, no detail specification, just mailing list. Add yourself to this list and wait... :) I guess it worths!
